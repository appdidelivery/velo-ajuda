## O que é o Upsell?
É a estratégia de oferecer produtos que combinam com o que o cliente já escolheu. Exemplo: sugerir "Carvão" para quem está comprando "Cerveja".

## Como Configurar
1. No cadastro de um produto, localize a secção **Compre Junto (Upsell)**.
2. O sistema listará os seus outros produtos cadastrados.
3. Selecione os itens que fazem sentido serem consumidos juntos.
4. Clique em **Salvar**.

## Visão do Cliente
Quando o cliente abrir o carrinho para finalizar o pedido, verá uma secção chamada **"Que tal pedir também?"** com fotos e preços dos itens sugeridos, permitindo adicioná-los com apenas um clique.

## Atalhos para Agilidade
- **Busca por Nome:** No campo de seleção de produtos, basta digitar as iniciais para filtrar instantaneamente.
- **Troco Automático:** Ao selecionar "Dinheiro", preencha o campo "Troco para" para que essa informação saia impressa na etiqueta do entregador.
- **Histórico:** O sistema salva os dados do cliente localmente para que, na próxima compra, você preencha o pedido ainda mais rápido.

> **Regra de Estoque:** Lembre-se que o lançamento manual também respeita o estoque. Se um produto estiver zerado, o sistema não permitirá o lançamento até que você atualize o saldo no menu **Estoque**.

## Criando seu Primeiro Cupom
No menu **Marketing**, localize a seção de **Cupons** e clique em **+ NOVO CUPOM**.

| Campo | O que definir |
| :--- | :--- |
| **Código** | O texto que o cliente vai digitar (Ex: BEMVINDO10). |
| **Tipo** | Escolha entre Desconto em **Porcentagem (%)** ou **Valor Fixo (R$)**. |
| **Valor** | O montante do desconto. |
| **Pedido Mínimo** | Valor que o carrinho deve atingir para o cupom ser válido. |

## Regras de Segurança
Para proteger seu lucro, você pode configurar:
- **Apenas Primeira Compra:** O cupom só funciona para novos clientes.
- **Limite de Uso:** Define quantas vezes o cupom pode ser usado no total ou por CPF.
- **Validade:** Data automática para o cupom expirar.

## Onde o cliente aplica?
O campo de cupom aparece dentro do **Carrinho**, logo antes do cliente escolher a forma de pagamento.

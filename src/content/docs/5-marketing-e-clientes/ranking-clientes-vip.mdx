## Como Funciona o Clube Fidelidade
O Clube Fidelidade da Velo não exige cadastro complexo. O sistema identifica o cliente pelo WhatsApp e acumula pontos automaticamente a cada pedido concluído.

## Configuração do Lojista
No menu **Marketing**, ative o **Clube Fidelidade** e defina:
1. **Regra de Pontos:** Quantos pontos o cliente ganha por R$ 1,00 gasto (Ex: 1 real = 1 ponto).
2. **Meta de Resgate:** Quantos pontos são necessários para ganhar o prêmio (Ex: 100 pontos).
3. **Texto do Prêmio:** O que o cliente ganha (Ex: "Ganhe uma Heineken gelada!").

## Ranking VIP
No menu **Clientes VIP**, você consegue visualizar quem são os seus melhores compradores.
- **Barra de Progresso:** Você vê exatamente quanto falta para cada cliente atingir o prêmio.
- **Resgate:** Quando o cliente atinge a meta, surge um botão verde de **Resgatar Prêmio** para você dar baixa no sistema após entregar o brinde.

## Visão do Cliente
O cliente acompanha a pontuação em tempo real através de uma barra de progresso no topo do App da sua loja.

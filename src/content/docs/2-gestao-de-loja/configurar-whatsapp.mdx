title: "Configurar Recebimento via WhatsApp"
description: "Saiba como configurar o número que receberá as notificações e pedidos."
---

## Importância do WhatsApp
Embora a Velo processe o pedido, o WhatsApp é o seu canal direto de comunicação e segurança. Pedidos feitos com pagamento em "Dinheiro" ou "Cartão no Motoboy" geram uma mensagem estruturada enviada diretamente para si.

## Passo a Passo
1. Aceda ao menu **Loja**.
2. Localize o campo **WhatsApp para Receber Pedidos**.
3. Insira o número com o código do país e DDD (Ex: 5551999999999).
   - **Dica:** Não use espaços, traços ou parênteses.
4. Clique em qualquer área fora do campo para o sistema salvar automaticamente.

## Notificações de Status
Sempre que alterar o status de um pedido no painel (Ex: de "Pendente" para "Em Rota"), o sistema oferecerá um botão para disparar um aviso automático ao cliente, mantendo-o informado e reduzindo a ansiedade da espera.

O Velo Delivery é uma plataforma focada em performance (Growth). Por isso, oferecemos um Hub de Integrações nativo para você conectar as maiores ferramentas de marketing do mercado e rastrear 100% do comportamento dos seus clientes.

## Como Conectar

Acesse o menu **Integrações** no seu painel. Você verá "cards" para cada ferramenta disponível. Para ativar qualquer uma delas, basta clicar em **+ Conectar API** e colar o seu código de rastreio (ID).

### Ferramentas Suportadas:

* **Meta Ads (Facebook e Instagram):** Insira o ID do seu Pixel e o Token CAPI para rastrear os eventos de *AddToCart* (Adição ao Carrinho), *InitiateCheckout* e *Purchase* (Compra Finalizada).
* **Google Analytics 4 (GA4):** Insira sua tag "G-XXXX" para mapear de onde estão vindo os acessos, cidades com mais vendas e dispositivos utilizados.
* **Google Ads:** Cole seu ID e Label de conversão para avisar o Google Ads sempre que uma venda for concluída via tráfego pago.
* **Google Tag Manager (GTM):** Liberdade total para instalar ferramentas de mapas de calor, pop-ups de terceiros ou chats.
* **Google Merchant Center:** Ative para gerar automaticamente a URL do seu catálogo XML para as campanhas de Google Shopping.
* **WhatsApp API:** Campo reservado para conexão oficial com servidores da Meta para disparos transacionais.

**Dica:** Em cada card de integração, deixamos um botão de "Ajuda" (👉) que te leva direto para a página oficial da plataforma para você descobrir onde achar o seu ID.
